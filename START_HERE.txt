STEP 1: Create GitHub Repository
---------------------------------
1. Go to https://github.com
2. Click "+" ‚Üí "New repository"
3. Name: dostbin-chatbot
4. Make it PUBLIC
5. Don't add README or .gitignore
6. Click "Create repository"


STEP 2: Upload Files
---------------------
Option A (EASIEST):
   1. Double-click: 1_SETUP_GIT.bat
   2. Double-click: 2_PUSH_TO_GITHUB.bat
   3. Enter your GitHub username when asked
   4. Enter your Personal Access Token as password

Option B (Manual):
   1. Open UPLOAD_TO_GITHUB.txt
   2. Follow the step-by-step commands


STEP 3: Deploy to Streamlit Cloud
----------------------------------
1. Go to https://share.streamlit.io
2. Sign in with GitHub
3. Click "New app"
4. Select: dostbin-chatbot repository
5. Main file: app.py
6. In Advanced Settings ‚Üí Secrets, add:
   GROQ_API_KEY = "your_groq_api_key_here"
7. Click "Deploy"
8. Wait 2-5 minutes
9. Your chatbot is LIVE!
